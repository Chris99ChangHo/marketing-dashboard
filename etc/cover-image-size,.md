### A3 landscape cover/back-cover image pixel sizes with 8mm margins

-   Page size: A3 landscape = 420×297mm.[](https://www.adobe.com/uk/creativecloud/design/discover/a3-format.html)
    
-   Content area after 8mm margins on all sides: 404×281mm.
    
-   Conversion: inches = mm/25.4; pixels = inches×DPI.[](https://pixelcalculator.com/en/index.php)
    
-   Recommended pixel dimensions for the 404×281mm content box:
    
    -   150DPI: 2,386×1,659px.
        
    -   200DPI: 3,181×2,213px.
        
    -   300DPI: 4,772×3,319px.
        
    -   350DPI: 5,567×3,872px.
        
    -   400DPI: 6,362×4,425px.
        
-   Practical recommendation:
    
    -   Use 300DPI for print-quality covers: 4,772×3,319px.
        
    -   If file size must be smaller, 200DPI (3,181×2,213px) is acceptable with some quality trade-off.
        
-   Reminder:
    
    -   The PDF’s physical size is governed by the template’s mm units; ensure images are generated to match the mm-defined content box for best clarity.
        

### Notes on A3 full-page pixels (for reference)

-   A3 at 300DPI (full page, no margins) ≈ 3,508×4,961px (portrait) or 4,961×3,508px (landscape).
